
# 📊 Data Asset
프로젝트 수행에 필요한 데이터셋과 분석 리소스를 관리합니다.

### 📂 데이터 관리 규칙
* **이미지 포맷**: 가능한 `.jpg` 또는 `.png` 형식을 사용해 주세요.
* **용량 제한**: 개별 파일당 1MB를 넘지 않도록 최적화해서 업로드 바랍니다. (대용량은 별도 클라우드 활용)
* **파일명**: `유형_번호.extension` (예: `slim_01.jpg`, `cooltone_01.png`)

### ⚠️ 주의사항
* 개인정보(얼굴 등)가 포함된 이미지는 반드시 비식별 처리를 하거나 동의를 얻은 데이터만 공유해 주세요.
