# 👔 StyleFit
> **"데이터로 찾는 나만의 핏, 사진 한 장으로 완성되는 정밀 큐레이션"**

전문가의 안목을 데이터화하여 사용자의 체형과 컬러에 최적화된 스타일을 제안하는 AI 엔진입니다.

---

### 💡 Project Vision
* **기획 배경:** 한국 패션 시장의 높은 미적 기준과 글로벌 브랜드의 로컬라이징 수요 반영
* **기획 의도:** 전문가를 직접 찾아가는 번거로움과 분석 후 제품 탐색의 막막함을 기술로 해결
* **핵심 목표:** 이미지 분석과 방대한 데이터 학습을 통한 디지털 컨설팅 엔진 구현

### 🛠 Tech Stack
* **Core Engine:** Python, PyTorch, MediaPipe
* **Cloud & DevOps:** Azure Machine Learning
