> Chrome 또는 Edge 사용을 권장합니다.

## 📋 사용 방법

### 1. 카메라 권한 허용
브라우저에서 카메라 접근 권한을 요청합니다. **반드시 "허용"을 눌러주세요.**  
권한을 거부하면 카메라가 작동하지 않습니다.  
**핸드폰 카메라 기능 추가 (전면, 후면 카메라 둘 다 가능)** 

### 2. 카메라 선택 후 시작
드롭다운에서 사용할 카메라를 선택하고 **카메라 시작** 버튼을 클릭합니다.

### 3. 가이드라인 안에 위치 잡기
화면에 표시된 사각형 가이드라인 안에 **얼굴부터 발끝**까지 들어오도록 위치를 조정합니다.

### 4. 자동 촬영
자세가 2초 이상 안정적으로 유지되면 카운트다운(5→1)이 시작되고, 자동으로 측정이 완료됩니다.

### 5. 결과 확인 및 저장
측정 결과(신장, 어깨, 골반 — 픽셀 단위)가 화면에 표시됩니다.  
**CSV 다운로드** 버튼으로 결과를 저장할 수 있습니다.

## 📐 촬영 권장 환경

| 항목 | 권장 조건 |
|---|---|
| 기기 | MS에서 제공해준 노트북, 제 핸드폰 기준 테스트 완료 |
| 카메라 각도 | 약 85도 (거의 수직) |
| 거리 | 카메라로부터 약 2m |
| 배경 | 단색의 밝은 배경 권장 |

> 기기 노트북, 스마트폰, 외장 웹캠 등 인터넷과 카메라가 접속할 수 있는 환경에서는 모두 가능
> **핸드폰 카메라 기능 추가 (전면, 후면 카메라 둘 다 가능)**  
> 기기에 따라 최적 거리와 각도가 달라질 수 있습니다.  
> 본인 환경에 맞게 조금씩 조정해 주세요.
> 카메라 변경 시 새로고침 후 다시 선택해주세요.

## ⚠️ 주의사항

- 측정값은 **픽셀(px) 단위**이며, 실제 cm/inch 수치가 아닙니다.
- **측정 정확도는 조명 환경, 카메라 해상도, 촬영 거리에 따라 달라질 수 있습니다.**
- 재측정이 필요한 경우 **재측정** 버튼을 클릭하세요.
- 결과 값이 나오면 CSV 파일 다운로드 가능합니다.
