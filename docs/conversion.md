# 🚀 StyleFit AI Master Engine v3.12.1

본 프로젝트는 단일 정면 사진을 기반으로 사용자의 신체 치수를 정밀하게 추출하고, 의류 사이즈 매칭에 필요한 핵심 데이터를 생성하는 AI 엔진입니다.

---

### 1. 📏 픽셀-cm 단위 환산 로직 (Unit Conversion)
사진 속의 픽셀 데이터를 실제 물리 단위(cm)로 변환하기 위해 **'정수리 보정(Head Offset)'** 기술을 적용합니다.

* **한계 극복**: AI가 인식하는 최상단점인 '코(Nose)'를 기준으로 삼을 경우 발생하는 키 손실을 보정합니다.
* **7% 법칙**: 인체 해부학 통계에 따라 정수리부터 코까지의 길이를 전체 키의 7%로 간주합니다.
* **환산 공식**:
    * **Available_Height_cm** = User_Height × 0.93
    * **cm/px** = Available_Height_cm / |Nose_y - Ankle_y| (Pixel)

---

### 2. 📍 지능형 허리라인 특정 (Waistline Mapping)
단순한 관절 트래킹을 넘어, 실제 의류 피팅의 기준이 되는 허리 위치를 찾아냅니다.

* **수직 보정 (Height Adj)**: AI가 찾은 골반 관절(23, 24번) 대비 실제 패션 허리라인(배꼽 부근)까지의 높이 차이를 미세 조정합니다.
* **수치 실시간 연동**: 허리라인 높이를 조정함에 따라 해당 지점의 **허리 너비, 상체 기장, 다리 길이** 수치가 실시간으로 재산출됩니다.
* **데이터 동기화**: 1cm의 높이 변화당 너비 수치를 약 1%씩 가감하여 인체의 굴곡을 수학적으로 반영합니다.

---

### 3. 📊 신체 부위별 측정 및 보정 근거
모든 수치는 **Size Korea(국가공인 인체 표준 데이터)** 및 의류 제작 실무를 기반으로 보정됩니다.

| 항목 | 측정 방식 | 보정 근거 (Offset/Ratio) |
| :--- | :--- | :--- |
| **어깨 너비** | 양측 견봉(Acromion) 간 거리 | **+1.5cm**: 삼각근(근육) 두께 반영 |
| **허리 너비** | 보정된 허리라인 횡단 거리 | **+4.0cm**: 옆구리 살집 및 피팅 여유분 |
| **어깨 둘레** | 어깨 너비 기반 입체 추정 | **x 2.3**: 한국인 표준 상체 두께 계수 |
| **허리 둘레** | 허리 너비 기반 입체 추정 | **x 2.7**: 한국인 표준 하체 전후 두께 계수 |
| **상체 기장** | 어깨 중앙 ~ 허리선 중앙 | **수직 거리**: 실제 상의 총장 비교 데이터 |

---

### 4. 💡 주요 기능 (Key Features)
* **Dual Unit System**: 모든 수치를 cm와 inch(인치)로 동시 표기하여 사용자 편의성 극대화
* **Visibility Control**: 분석 선 및 수치 텍스트의 실시간 온오프(On/Off) 기능을 통한 가시성 확보
* **Body Type Analysis**: 어깨와 허리의 비율을 분석하여 체형 유형(역삼각형, 균형형 등) 자동 판정
* **Confidence Score**: AI 인식 결과의 신뢰도를 실시간으로 산출하여 측정 정확도 보장
