# 💻 Source Code Repository
이 폴더는 프로젝트의 실제 구현 코드가 관리되는 공간입니다.

### 📝 코딩 규칙 (Coding Convention)
* **언어**: Python 3.1 기반
* **파일명**: `기능명_담당자.py` (예: `body_analysis_택근.py`)
* **주석**: 코드 상단에 어떤 기능을 수행하는지 반드시 한 줄 주석을 달아주세요.

### 🚀 업로드 가이드
* 작업한 코드는 각자의 기능을 확인한 후 업로드해 주세요.
* 공통으로 사용하는 함수는 `utils.py`를 만들어 관리할 예정입니다.
